## 特徴

<CardGrid stagger>
  <Card title="データベース駆動のエージェントグラフ" icon="setting">
    AIワークフローをコードではなく行として定義。再デプロイなしで組織ごとに処理パイプラインを切り替え可能。
  </Card>
  <Card title="GenUIチャットインターフェース" icon="puzzle">
    AIはテキストで返答するだけではない。カード、ギャラリー、インタラクティブコンポーネントなど、実際のFlutterウィジェットをJSONからランタイムで生成。
  </Card>
  <Card title="オンデバイスGemma" icon="laptop">
    インテント解析はネットワークに到達する前にスマートフォン上でローカル実行。デフォルトでプライバシー重視。
  </Card>
  <Card title="CLIPベクトル検索" icon="magnifier">
    画像とその説明文を同一の768次元空間に埋め込み。画像で検索、テキストで検索、雰囲気で検索。
  </Card>
  <Card title="ビジュアルワークフロービルダー" icon="pencil">
    Reactダッシュボードでプロジェクト作成、デバイス登録、APIキー発行、エージェントグラフのドラッグ＆ドロップを一か所で実行。
  </Card>
  <Card title="リアルタイム運用ビュー" icon="bolt">
    アップロード、説明文生成、セグメンテーション、実行ステップの進行に合わせてDocumentsとRunsが更新。
  </Card>
  <Card title="MCPツールがファーストクラス" icon="open-book">
    説明文生成、CLIPエンベディング、類似検索、セグメンテーションモデルをMCP経由で公開。エージェントも人も利用可能。
  </Card>
</CardGrid>

## 技術スタック

| レイヤー | 技術 | 役割 |
|---|---|---|
| **クライアント** | Flutter, Dart, flutter_gemma, GenUI, fpdart | カメラキャプチャ、オンデバイスLLM、AI生成UI、関数型エラーハンドリング |
| **API** | Bun, Hono, better-auth, Inngest, Pino | RESTルート、署名付きアップロード、ジョブスケジューリング、構造化ログ |
| **ダッシュボード** | React 19, TanStack Router, Tailwind, shadcn/ui | ワークフロービルダー、プロジェクト/デバイス/APIキー管理、ライブ運用UI |
| **エージェント** | Go, Gin, LangGraph, LangChain, inngestgo | グラフベースのエージェントオーケストレーション、ReActワーカー、ステップレベルトレーシング |
| **MCP** | Go, MCP go-sdk, replicate-go, GORM | CLIPエンベディング、説明文生成、セグメンテーション、類似検索 |
| **ストレージ** | Postgres 18 + pgvector, S3互換, Redis | ベクターインデックス、オブジェクトストレージ、セッションキャッシュ |
